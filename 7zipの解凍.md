# 7zipの解凍方法
- 実験システムは研究室のboxの以下の場所に7zip形式で圧縮されている
  - 各条件のシステムのプロト版：「k-lab/docs/2025archive/2025中村勇理_アーカイブ/実験システム/Proto」
  - 各参加者のシステム
    - セルフ条件：「「k-lab/docs/2025archive/2025中村勇理_アーカイブ/実験システム/セルフ」
    - フロイト条件：「k-lab/docs/2025archive/2025中村勇理_アーカイブ/実験システム/フロイト」
    - 友人条件：「k-lab/docs/2025archive/2025中村勇理_アーカイブ/実験システム/友人」
- 7zipファイルの解凍には「7zip File Manager」が必要
  - [こちらのサイト](https://7-zip.opensource.jp/)からダウンロード
